Amigo Fiel

Portal de Adoção de Pets — Transformando vidas uma pata de cada vez.

Público (Consumidor)

GET /api/pets

Lista todos os pets disponíveis. Suporta filtros por nome, espécie e porte.

Parâmetro Tipo Descrição
name String Nome ou parte do nome (case-insensitive).
species String Cachorro, Gato, Ave, Roedor, Outro.
size String Pequeno, Médio, Grande.
GET /api/pets/:id

Detalhes específicos de um pet.

Administração (Protegido)

Requer Header: Authorization: Bearer <token>

POST /api/admin/pets

Cadastra um novo amigo no portal.

PUT /api/admin/pets/:id

Atualiza dados de um pet existente.

DELETE /api/admin/pets/:id

Remove o pet do sistema.
